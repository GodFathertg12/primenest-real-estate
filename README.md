# PrimeNest Real Estate Platform

A premium real estate frontend experience built with vanilla JavaScript, focused on high-conversion UI, property storytelling, and dynamic filtering.

## Features
- Property filtering (location, price, type)
- Dynamic property pages
- Image gallery system
- Contact form with validation
- Theme switching (dark/light)

## Tech
- Vanilla JavaScript (modular architecture)
- CSS
- Local state management

## Goal
Built as a portfolio project to demonstrate frontend architecture and product thinking.
